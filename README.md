# TodoRodPipy

TodoRodPipy is a project in React like Pipefy app faker.

## Installation

Use the package manager [yarn](http://yarnpkg.com) or [npm](http://npmjs.com) to install GetCar.


```bash
  yarn install # or npm install
```

## Usage

```bash
  yarn start
```
